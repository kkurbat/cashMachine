## Сash Machine , техническое задание для java интенсива от Senla
Java SDK ver 12,Project language level - 8 Lambdas, type annotations etc.

1. Реализованно консольное меню для приложения (ввод с консоли);
2. Создан бат файл(cashMachine.bat);
3. При неправильном введение пароля 3 раза,карта блокируется;
4. Блокировка с карты снимается автоматически, по истечению суток;
5. Все карты хранятся в текстовом файле по пути(src/main/resources/data/cards.txt),где в первой строке указывается баланс банкомат, а
в последующих id карты, баланс, зашифрованный пароль, статус карты, последняя дата блокировки;
6. Расшифрованные пароли находятся в файле encodedPINS.txt(src/main/resources/data/encodedPINS.txt);
7. После окончания работы, программа сохраняет свое состояние;
8. Реализованны тесты и логирование.
